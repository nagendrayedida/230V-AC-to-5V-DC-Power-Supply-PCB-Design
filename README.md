#230V AC to 5V DC Power Supply PCB Design

#Description:
This project is a PCB design that converts **230V AC to 5V DC** using KiCad. It includes schematic design, PCB layout, routing, and ground plane implementation.

#Features include:
* 2-layer PCB design
* Proper component placement and routing
* Ground plane for stability
* Designed with high-voltage isolation

#Block Diagram:
230V AC → Rectifier → Filter → Regulator → 5V DC Output.

#Files Included:
* Schematic
* PCB Layout
* Gerber & Drill Files
* Images

#Applications:
Embedded systems, IoT devices, low-power electronics.
